# Team-division（チーム分けアプリ）

メンバーを登録して、指定した人数（または指定したチーム数）でランダムにチーム分けできるWebアプリです。

## 2つのモード（上部タブで切替）

- **🎰 ルーレット抽選** — 人数とチーム数（と役割）を設定してスタート。「発表！」を押すと1人ずつルーレット演出でチーム・役割を発表し、「次の人へ」で順番に進みます。発表前は「？」表示なので**前の人の結果は次の人に見えません**。最後に結果一覧を表示
- **📝 リストで分ける** — 名前を登録して、人数/チーム数を指定して一括チーム分け

## 特徴

- **ビルド不要** — `index.html` をブラウザで開くだけで動作します
- **メンバー登録** — 名前を1人ずつ、または改行・カンマ・スペース区切りでまとめて追加できます
- **自動保存** — 登録したメンバーはブラウザの localStorage に保存され、次回も残ります
- **保存リストで使い回し** — メンバーに名前を付けて複数のリストとして保存し、いつでも呼び出して使い回せます
- **2つの分け方**
  - 1チームの人数で指定（例: 4人ずつ）
  - チーム数で指定（例: 3チームに分ける）
- **役割のランダム割り当て** — 書記・発表・司会・タイムキーパーなどの役割を各チームにランダムで割り当て。役割は自由に追加・削除でき、人数が役割より少ないチームでは1人が複数役割を兼任します
- **均等配分** — 割り切れない場合も人数差が最大1人になるように振り分けます
- **再シャッフル** — 同じ設定でワンクリックで分け直せます
- **音声読み上げ** — ルーレットの発表結果をブラウザの音声合成で読み上げ（画面の「🔊 結果を音声で読み上げる」でON/OFF）
- **チームメモ（5W1H）** — 結果一覧の下に、チームごとに番号付きの 5W1H（いつ／どこで／だれが／なにを／なぜ／どうやって＋備考）メモを枠線付きで記録。内容は端末の localStorage に自動保存。チーム番号を入力して目的のメモへすぐ移動できます
- **CSV書き出し** — 各チームのメモを CSV で出力。別システム（AI）での要約・整理を想定し、**1行＝1項目**の long 形式（`team_number, team_name, members, element_key, element_label, content`）＋ UTF-8 BOM 付き（Excelでも文字化けしません）

## メモのCSV構成（AI連携を想定）

CSV は LLM が解釈しやすいよう、1行に1つの 5W1H 項目を持つ正規化（long）形式です。

| 列 | 内容 | 例 |
| --- | --- | --- |
| `team_number` | チーム番号 | `1` |
| `team_name` | チーム名 | `Aチーム` |
| `members` | メンバー（` / ` 区切り） | `山田 / 田中 / 佐藤` |
| `element_key` | 5W1Hの英語キー | `when` / `where` / `who` / `what` / `why` / `how` / `notes` |
| `element_label` | 画面表示ラベル | `いつ (When)` |
| `content` | 入力された内容 | `来週月曜の朝礼で` |

別端末との完全なリアルタイム同期は行いません（バックエンド不要の静的サイトのため）。メモは各端末の localStorage に保存され、CSV で受け渡し・集約できます。

## 使い方

1. `index.html` をブラウザで開く（または公開URLを開く）
2. **🎰 ルーレット抽選**: 人数・チーム数・役割を設定して「スタート」→「発表！」で1人ずつ発表
3. **📝 リストで分ける**: 名前を登録 →「ルーレットでスタート」でチームごとに発表
4. 結果一覧の下の「チームメモ（5W1H）」に話し合いを記録
5. 「⬇ CSVで書き出す」で各チームのメモを出力
